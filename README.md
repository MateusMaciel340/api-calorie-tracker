## How to run the project? 

The steps to access the project are very simple, but follow in the following order:

```bash
npm install
```

Configuring your file (.env) or dotenv

```bash
PORT_SERVER=8080

DB_HOST=localhost
DB_USER=azizbek
DB_DATABASE=calorie
DB_PASSWORD=12345
DB_PORT=5432
```
